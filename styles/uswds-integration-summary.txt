QUICK START
================================================================================

1. Include CSS in your HTML (head.html):
   <link rel="stylesheet" href="/styles/uswds-compiled.css">

2. Use USWDS classes in your HTML:
   <button class="usa-button">Click Me</button>
   <div class="grid-container padding-4">Content</div>

3. Rebuild after customizations:
   npm run build:uswds

================================================================================
UPDATING USWDS (EASY!)
================================================================================

1. Check for updates:
   npm outdated

2. Update USWDS:
   npm update uswds

3. Review changelog:
   https://github.com/uswds/uswds/releases

4. Rebuild:
   npm run build:uswds

5. Test your site

6. Deploy
